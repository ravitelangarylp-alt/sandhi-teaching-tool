ಸೂಚನೆ (Important): ಯಣ್ ಸಂಧಿಯಲ್ಲಿ ಕೇವಲ ಪೂರ್ವಪದದ ಅಂತ್ಯದಲ್ಲಿರುವ 'ಇಕ್' (इ/उ/ऋ/ऌ) ಮಾತ್ರ ಬದಲಾಗುತ್ತದೆ. ಪರಪದದ ಆದಿಯಲ್ಲಿರುವ 'ಅಚ್' (ಸ್ವರ) ಹಾಗೆಯೇ ಉಳಿದು, ಹೊಸದಾಗಿ ಬಂದ ಅರ್ಧಾಕ್ಷರಕ್ಕೆ ಮಾತ್ರೆಯಾಗಿ ಸೇರಿಕೊಳ್ಳುತ್ತದೆ.
इ / ई + अच् (ಸ್ವರ) ➔ य्
ಸ್ವರ ಸೇರಿದಾಗ: य् + अ = य य् + उ = यु य् + आ = या
उ / ऊ + अच् (ಸ್ವರ) ➔ व्
ಸ್ವರ ಸೇರಿದಾಗ: व् + अ = व व् + इ = वि व् + ए = वे
ऋ / ॠ + अच् (ಸ್ವರ) ➔ र्
ಸ್ವರ ಸೇರಿದಾಗ: र् + अ = र र् + उ = रु र् + आ = रा
ऌ + अच् (ಸ್ವರ) ➔ ल्
ಸ್ವರ ಸೇರಿದಾಗ: ल् + अ = ल ल् + आ = ला

ಅಸ್ಯಾಂ ಕಥಾಯಾಂ ಯತ್ರ ಯಣ್ಸಂಧಿಃ ಭವಿತುಮ್ ಅರ್ಹತಿ ತಾದೃಶೇಷು ಸ್ಥಾನೇಷು ಯಣ್ ಸಂಧಿಂ ಕಿಂ ಸ್ಥಾನೇ ಅಧಃ ಲಿಖತ — ಈ ಕಥೆಯಲ್ಲಿ ಎಲ್ಲಿ ಯಣ್‌ ಸಂಧಿ ಆಗಬೇಕಿತ್ತು ಎಂಬ ಸ್ಥಳಗಳನ್ನು ಗುರುತಿಸಿ, ಯಣ್ ಸಂಧಿ ಮಾಡಿ ಕೆಳಗೆ ಬರೆಯಿರಿ.
कश्चन कृषिकः आसीत्। तस्य चत्वारः पुत्राः आसन्। ते चत्वारः अपि अलसाः आसन्। ते प्रतिदिनं दधि इत्यादि खादित्वा स्थूलाः अभवन्। एकस्मिन् दिने सः कृषिकः रुग्णः अभवत्। सः इतः परं न अहम् जीवामि इति ज्ञातवान्। उपायेन पुत्राणाम् आलस्यं दूरीकरोमि अहम् इति अचिन्तयत्। एकस्मिन् दिने पुत्रान् आहूय एवम् उक्तवान् मम कृषिक्षेत्रे अनेकानि अमूल्यानि रत्नानि सन्ति, मम मरणानन्तरं स्वीकुर्वन्तु इति केषुचित् दिनेषु सः मृतः। अभवत्। पुत्राः पितुः उक्तिम् अनुसरन्तः धनस्य आशया कृषिक्षेत्रस्य खननं कृतवन्तः। किन्तु एकम् अपि रत्नं न प्राप्तवन्तः। तावता वर्षकालः आरब्धः। वर्षकालस्य समाप्तेः अनन्तरम् अन्विष्याम इति निश्चितवन्तः। क्षेत्रे खननं तु कृतम् आसीत् इति कारणात् धान्यानां बीजानि उप्तवन्तः। समृद्धरूपेण धान्यानि उत्पन्नानि। तानि धान्यानि विक्रिय बहु धनं प्राप्तवन्तः। धनं दृष्ट्वा ज्ञातवन्तः यत् अस्माकं पिता उक्तानि रत्नानि एतानि एव इति। ततः परम् आलस्यं विहाय सन्तोषेण जीवितवन्तः।
