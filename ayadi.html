ವರ್ಣ-ಪರಿವರ್ತನೆ (Transformation)
ಸೂಚನೆ (Important): ಅಯಾದಿ ಸಂಧಿಯಲ್ಲಿ ಪೂರ್ವಪದದ ಅಂತ್ಯದಲ್ಲಿರುವ 'ए', 'ओ', 'ऐ', 'ಔ' ಎಂಬ ಸ್ವರಗಳು ಪರಪದದ ಆದಿಯ ಸ್ವರದ ಮುಂದೆ ಬಂದಾಗ, ಅವು ಕ್ರಮವಾಗಿ 'अय्', 'अव्', 'आय्', 'आव्' ಎಂಬ ಅರ್ಧಾಕ್ಷರಗಳಾಗಿ ಪರಿವರ್ತನೆ ಹೊಂದುತ್ತವೆ.
ए (e) + स्वर (Vowel) ➔ अय्
ಸ್ವರ ಸೇರಿದಾಗ: अय् + अ = अय अय् + उ = अयु अय् + आ = अया
ओ (o) + स्वर (Vowel) ➔ अव्
ಸ್ವರ ಸೇರಿದಾಗ: अव् + अ = अव अव् + इ = अवि अव् + ए = अवे
ऐ (ai) + स्वर (Vowel) ➔ आय्
ಸ್ವರ ಸೇರಿದಾಗ: आय् + अ = आय आय् + उ = आयु आय् + आ = आया
औ (au) + स्वर (Vowel) ➔ आव्
ಸ್ವರ ಸೇರಿದಾಗ: आव् + अ = आव आव् + इ = आवि आव् + ए = आवे

सूत्रम् : लोपः शाकल्यस्य (६.१.१२७)

अर्थः (Meaning): According to the grammarian Shakalya, when a vowel (ac) is followed by another vowel, the preceding vowel is completely elided (dropped / lopa).

कुत्र (Applicability): This acts as an exception (apavada) to the normal Yan (इ/उ/ऋ/ऌ → य्/व्/र्/ल्) and Ayadi (ए/ओ/ऐ/औ → अय्/अव्/आय्/आव्) sandhi rules. Instead of substituting the vowel, the vowel simply disappears. The textbook has purposely left the space to show 'this is the exact result of Lopa'.

अभ्यास ६.८ रल्लिरुव उदाहरणगल्लि इदु हेगे कॆलस माडुत्तदे:

सन्धिपद विभजन स्पष्टीकरण
कस्मा इदम्कस्मै + इदम्लोपः ऐ-वर्णस्य (ऐ completely elided)
त आगताते + आगतालोपः ए-वर्णस्य (ए completely elided)
सून आगच्छसूनु + आगच्छलोपः उ-वर्णस्य (उ completely elided)
साध अत्रसाधु + अत्रलोपः उ-वर्णस्य (उ completely elided)
हर एहिहरे + इहिलोपः ए-वर्णस्य (ए elided), ततः अ+इ = ए (Guna applies)
टिप्पणी (Note): In some cases (e.g., का आगती) the simple Dirgha Sandhi (आ+आ = आ) rule applies. Because लोपः शाकल्यस्य is an optional (विकल्पः) rule.

अभ्यासः ६.९ गद्य-संधि-विच्छेदः
अस्यां कथायां यत्र अयादि सन्धिः भवितुमर्हति तादृशेषु स्थानेषु अयादि सन्धिं रिक्तस्थाने लिखत
ಈ ಕಥೆಯಲ್ಲಿ ಎಲ್ಲಿ ಅಯಾದಿ ಸಂಧಿ ಆಗಬೇಕಿತ್ತು ಎಂಬ ಸ್ಥಳಗಳನ್ನು ಗುರುತಿಸಿ, ಕೆಳಗಿರುವ ಖಾಲಿ ಜಾಗಗಳಲ್ಲಿ ಸಂಧಿ ಮಾಡಿ ಬರೆಯಿರಿ.
कथन ग्रामः आसीत्। तत्र द्वौ माजारौ आस्ताम्। तौ द्वौ अपि मित्रे आस्ताम्। ग्रामे आदिनं मिलित्वा क्रीडतः स्मः। तयोः एकस्मिन् दिने एका रोटिका लब्धा। तौ द्वौ अपि बहु बुभुक्षितौ आस्ताम्। अतः द्वौ अपि अहं रोटिकां खादिमि इति कोलाहलं कृतवन्तौ। तदा एकः वानरः आगतवान्। कस्मै इयं रोटिका इति अहं निर्णयं करोमि इति उक्तवान्। तौ अङ्गीकृतवन्तौ। सः एकां तुलाम् आनीतवान्। रोटिकां कर्तयित्वा एकस्मिन् पार्श्वे एकां भागम् अपरस्मिन् अन्यां भागं स्थापितवान्। माजारौ बुभुक्षितौ पश्यन्तौ आस्ताम्। तुला एकस्मिन् पार्श्वे अधः गता। सः वानरः ततः किञ्चित् रोटिकां निष्कास्य खादितवान्। तदा तुला अन्यस्मिन् पार्श्वे अधः गता। ततः किञ्चित् खादितवान् वानरः। एवं सम्पूर्णां रोटिकां वानरः खादितवान्। तौ माजारौ पश्चात्तापम् अनुभवन्तौ असन्तोषेण गतवन्तौ।

ಸಂಧಿಯನ್ನು ಎಲ್ಲಿ ಮಾಡಲೇಬೇಕು ಮತ್ತು ಎಲ್ಲಿ ಬಿಡಬಹುದು ಎಂಬುದರ ಬಗ್ಗೆ ವ್ಯಾಕರಣ ಶಾಸ್ತ್ರದಲ್ಲಿ ಒಂದು ಪ್ರಸಿದ್ಧವಾದ ಶ್ಲೋಕವಿದೆ:

ने + अनम् = नयनम्
२. धातूपसर्गयोः नित्या ಧಾತು-ಉಪಸರ್ಗಗಳ ನಡುವೆ (ಕಡ್ಡಾಯ)

नि + अवसत् = न्यवसत्

ಸಮಸ್ತ ಪದಗಳನ್ನು (ಸಮಾಸ) ರಚಿಸುವಾಗ ಸಂಧಿ ಕಡ್ಡಾಯ.
